How to Use

This tool helps you discover IoT devices, assess their exposure, and prioritize security risks using real-world vulnerability intelligence.

  • Start a Network Scan
    Click Run Network Scan to detect active devices on your local network. The analyzer identifies devices using open ports, exposed services, and network behavior.
  • Device Identification
    Each discovered device is fingerprinted to determine its type (camera, embedded web device, IoT gateway, sensor, etc.) based on protocol and service exposure.
  • Vulnerability Intelligence
    Detected services are correlated with known CVEs from the National Vulnerability Database (NVD) to identify real-world security weaknesses.
  • Risk Scoring
    Devices are assigned a risk score and categorized as:
    • Green — Low or no known risk
    • Yellow — Moderate exposure
    • Red — Critical vulnerabilities or exposed interfaces
  • Prioritize Action
    Devices marked Red may expose unsecured web interfaces, default credentials, or known exploits and should be addressed immediately.
  • Continuous Monitoring
    Re-run scans periodically to detect newly added devices, configuration changes, or emerging vulnerabilities.
